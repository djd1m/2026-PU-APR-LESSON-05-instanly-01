# INS-007: Telegram banned in RF — never reference as channel

**Status:** Active
**Created:** 2026-04-29
**Hits:** 0

## Context
User explicitly stated Telegram is banned in Russia for this project context.

## Solution
- Never mention Telegram as marketing channel, integration, or communication method
- Use alternatives: SEO (Yandex), YouTube, vc.ru, Habr, Yandex.Direct, email
- For messaging: WhatsApp, VK instead of Telegram
- Referral mechanics: email invite links, not Telegram bots

## Prevention
- Grep codebase for "telegram" before any release
- All marketing/growth docs must use web-only channels
