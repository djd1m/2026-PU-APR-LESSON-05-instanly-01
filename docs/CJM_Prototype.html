🚀 A: Турбо-Агентство

Сильный стартовый сегмент — агентства платят быстро и приводят клиентов. Но безлимит аккаунтов технически дорогой. Лучший вариант для быстрого cash flow.

→ Добавить в Phase 2 (месяц 4-6)
🤖 B: AI-Продажник РЕКОМЕНДОВАН

Самый большой сегмент (40%), AI-персонализация — главный тренд 2026. GPT/YandexGPT достаточно хороши для русского B2B. Максимальный WOM: «AI написал — клиент ответил».

→ Core MVP (месяц 1-3)
🛡️ C: Рост без спама

Самый защищённый moat (compliance + CRM), но медленный старт. 95% deliverability — инженерно сложно. AmoCRM/Bitrix24 интеграции трудоёмки.

→ Enterprise-фичи в Phase 3 (месяц 7-12)
